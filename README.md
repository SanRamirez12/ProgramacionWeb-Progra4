# Programming IV / Web Programming

This repository contains **in-class work, assignments, and projects** developed as part of the **Programming IV / Web Programming** course of the degree program.  
It serves as a collection of practical exercises and a complete web application project focused on backend and full-stack web development.

---

## Repository Overview

The purpose of this repository is to demonstrate the application of **web programming concepts**, including server-side development, database integration, and MVC architecture, through both incremental classwork and a final project.

---

## Repository Structure

- `ProyectoProgra4/`  
  Main course project. A web management system developed using **ASP.NET MVC**, **C#**, and **Entity Framework**, integrating database persistence, authentication, and business logic.

- `Tareas/`  
  Assignments and individual tasks completed throughout the course.

- `TrabajosClase/`  
  In-class exercises and practice files covering core web programming topics.

- `Pasos Entity Framework.txt`  
  Reference notes and setup steps related to **Entity Framework** configuration and usage.

- `ProyectoProgra4.rar`  
  Archived version of the main project for backup or submission purposes.

---

## Technologies and Concepts Applied

- ASP.NET MVC (Model–View–Controller architecture)  
- C# backend development  
- Entity Framework (ORM)  
- SQL Server integration  
- CRUD operations  
- Session management and authentication  
- Relational database design  
- HTML and CSS for web interfaces
